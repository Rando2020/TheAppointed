# Assets

The upstream `Rando2020/ProjectTactic` repo's `assets/` folder is currently
**unpopulated** ‚Äî every file under `assets/characters/`, `assets/environments/`,
`assets/ui/`, `assets/concept/`, etc. is either missing or a 128-byte stub placeholder
generated by the project's asset pipeline (see `ASSET_PIPELINE.md` in the source repo).

The `src/assets/` tree (icons, character portraits, tile textures) is the same ‚Äî every PNG
is a 128‚Äì129 byte placeholder that exists to assert filenames during integration. No actual
artwork has been delivered yet.

## What this means

- **No logos, character portraits, environment art, or icon sets are available.**
- The UI kit and preview cards therefore use **initial-glyph portraits** (a single letter
  drawn in Cinzel Decorative inside a sin-colored bordered tile) as a deliberate, on-brand
  placeholder ‚Äî *not* a final art direction.
- When real assets arrive, drop them in here under subfolders matching the upstream layout
  (`characters/`, `environments/`, `ui/`, `concept/`). Swap the `.portrait-lg` /
  `.party-card .portrait` / `.tile .unit` initial-block in `ui_kits/the-appointed/` for
  `<img>` references and the rest of the kit holds together unchanged.

## Folder layout (mirrors upstream `Rando2020/ProjectTactic` assets/)

- `characters/` ‚Äî sprites, portraits, class visuals, enemy sheets
- `environments/` ‚Äî maps, tiles, town backgrounds, combat arenas
- `ui/` ‚Äî icons, buttons, HUD elements, menus, status badges
- `concept/` ‚Äî moodboards, references, approved art direction
- `audio/` ‚Äî SFX, voice barks, UI feedback sounds
- `music/` ‚Äî loops, themes, stingers, exported stems

## ‚öÝ Flagged for the user

**This project needs real visual assets** before it can produce final-fidelity output. At
minimum:

1. **Logo / wordmark.** "The Appointed: As Above" ‚Äî ideally with a sacred-gold treatment
   of "As Above" set against the bone-white primary.
2. **Character portraits.** One per member of The Seven. The current placeholders are
   *intentionally* abstract; we cannot generate real ones.
3. **Antechamber art.** A single full-bleed reference of the hub for use as a title-screen
   backdrop. The CSS currently fakes this with radial gradients.
4. **Icon set.** Combat (sword, bow, staff, shield, heart), state (blessed, stunned,
   burning), and elements (fire, ice, thunder, water, holy, dark, earth). Closest CDN
   substitute that would fit the stone-and-iron aesthetic: **Tabler Icons** filled-and-stroked
   set, recolored to brass. Flagged as a substitution if used.
