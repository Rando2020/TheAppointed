# Static UI/UX Polish List

Priority is based on making the game addictive but readable before character asset production.

## P0 - Must Exist For Fun Testing

1. Battle command bar
   - Move, Attack, Ability, Item, Wait icons.
   - Clear selected/disabled/hover states.
   - Short action hint text.

2. Damage forecast panel
   - Expected HP damage.
   - Hit percent and crit percent.
   - Temper/Ether damage.
   - Facing and height modifiers.
   - Element weakness/resist label.

3. Enemy intent panel
   - Enemy name.
   - Intended action.
   - Target.
   - Expected damage.
   - Lethal warning state.

4. Tile info panel
   - Terrain name.
   - Move cost.
   - Height.
   - Hazard/status effect.
   - Tactical note.

5. Turn order sidebar
   - Upcoming unit portraits/tokens.
   - Team color.
   - Current turn highlight.
   - Speed/CT hint if available.

6. Range and selection markers
   - Selected tile.
   - Move range.
   - Attack range.
   - Ability range.
   - Blocked tile.
   - Objective tile.

7. Battle spoils screen
   - Gold, JP, item cards.
   - "Why this matters" stat deltas.
   - Continue button.

8. HP carryover display
   - Show current HP after battle.
   - Show small auto-recovery amount.
   - Show wounded/downed units.
