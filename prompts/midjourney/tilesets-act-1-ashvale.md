# Act 1 Ashvale Tile Prompts

Use these prompts for the first cohesive Midjourney terrain batch. The visual target is a lavender-stone woodland approach: pink flowering trees, dry gold grass, purple wildflowers, slate cliffs, old shrine paths, and readable isometric tactical tiles.

## Global Style Anchor

Append this style block to each prompt:

```txt
overhead isometric tactical RPG battle map tile, romantic dark fantasy woodland, lavender slate stone, pink flowering trees, dry golden grass, purple wildflowers, hand-painted anime realism, inked linework, watercolor texture, premium tactics game readability, clean material silhouette, soft cinematic dawn light, original fantasy art, no characters, no text, no UI, no watermark
```

## Midjourney Settings

Use these settings for individual terrain tiles:

```txt
--ar 3:2 --v 6.1 --stylize 150 --chaos 8
```

Use these settings for larger reference boards:

```txt
--ar 1:1 --v 6.1 --stylize 180 --chaos 10
```

## Acceptance Notes

- Prefer clean diamond silhouettes over scenic full-map compositions.
- Reject tiles with characters, large trees covering the center, text, UI, or heavy cast shadows.
- Keep the walkable center readable.
- Keep each terrain type identifiable at small size.
- Export best candidates into `godot/assets/ui/tiles/` for current battle rendering, or `godot/assets/tiles/` for the registry-backed terrain pack.
