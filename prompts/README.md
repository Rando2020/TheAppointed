# Prompt Library

Store reusable AI prompts here so assets and design ideas can be regenerated consistently.

## Pathways

- `midjourney/` - environment, character, UI, tileset, icon, and portrait prompts.
- `suno/` - battle themes, town themes, boss themes, stingers, leitmotifs.
- `character-design/` - character sheets, class outfits, enemy variants.
- `environment-design/` - towns, ruins, battle maps, lighting, biome art direction.
- `ui-design/` - menus, HUD, skill buttons, status icons, battle timeline.

## Naming convention

Use kebab-case with the target asset in the name.

Example: `midjourney/guardian-ignareth-boss-sprite.md`
