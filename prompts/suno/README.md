# Suno Prompts

Use this folder for music and audio prompt recipes.

Recommended categories:

- `battle-theme-*.md`
- `boss-theme-*.md`
- `town-theme-*.md`
- `victory-stinger-*.md`
- `guardian-leitmotif-*.md`

Always include:

- intended scene or system
- emotional target
- tempo range
- instrumentation
- loop notes
- banned references or copyright-sensitive wording to avoid
