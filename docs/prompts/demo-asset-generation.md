# Demo Asset Generation Prompts

These prompts define the locked visual direction for ProjectTactic's first playable vertical slice.

## Global art direction

Use this style language consistently:

- Reimagined emberlit HD-2D tactical diorama.
- Final Fantasy Tactics inspired tactical readability.
- Vagrant Story inspired darker war atmosphere.
- Fire Emblem political-war mood.
- Watercolor texture overlays.
- UE5-level cinematic lighting references.
- Isometric top-down tactical readability.
- Anime realism hybrid.
- Painterly but gameplay readable.
- Original designs only.
