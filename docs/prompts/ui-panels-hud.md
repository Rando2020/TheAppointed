# UI Panels & HUD Prompts

This file contains prompts for generating the user interface chrome used throughout the battle HUD.  The aesthetic is dark stone with muted metallic accents, invoking a grounded fantasy atmosphere.  Elements should be cohesive and readable at small sizes.

## Panel Backgrounds
*Create a set of rectangular panel backgrounds carved from dark stone.  Each panel should have beveled edges, subtle engraved patterns, and a hint of metallic trim.  Provide several sizes suitable for command bars, status displays, and information pop-ups.*

## Command Bar
*Generate a horizontal bar with slots for five command buttons.  The bar should match the stone panel style and include dividers between each button slot.  Leave space below for tooltips.*

## Turn Order Sidebar
*Design a vertical sidebar panel that displays a list of upcoming turn icons.  The sidebar should have a column of slots framed by stone and accented with subtle metallic highlights.  Provide placeholders for character portraits and small CT numbers.*

## Tile Highlight Diamonds
*Create diamond-shaped highlight markers that overlay tiles to indicate selection and reachable areas.  Produce variants in blue (movement range), red (attack range), and yellow (special ability range).  The diamonds should have thin glowing borders.*

## HP/Temper/Ether Bars
*Generate narrow horizontal bars for representing Health (HP), Temper, and Ether.  Each bar should have a dark stone frame with a colored fill: green for HP, orange for Temper, blue for Ether.  Include subtle tick marks and a small icon at the left end of each bar.*

## Command Button Icons
*Design five unique circular icons for the battle commands: Move, Attack, Ability, Item, Wait.  Each icon should fit within a square slot on the command bar and feature minimalistic line art that is easily recognizable.  Use light metallic tones against a darker background for contrast.*
