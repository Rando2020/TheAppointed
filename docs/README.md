# Documentation Hub

This folder is the design and production brain for ProjectTactic / Vaelthar.

## Pathways

- `lore/` - worldbuilding, factions, guardians, places, terminology.
- `mechanics/` - combat rules, jobs, stats, progression, tactical systems.
- `characters/` - playable units, enemies, bosses, biographies, portraits, voice notes.
- `production/` - roadmap, backlog notes, release planning, testing notes.

## Rule

When a gameplay idea becomes real code, document the design intent here first or update the matching mechanic file in the same pull request.
