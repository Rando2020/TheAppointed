# Characters

Use this folder for playable units, enemies, bosses, portraits, relationship notes, and voice/personality direction.

## Suggested files

- `party.md` - playable heroes, classes, arcs, combat roles.
- `enemies.md` - common enemy families and tactical behavior.
- `bosses.md` - Guardian bosses, Omega Null, major antagonists.
- `dialogue-style.md` - tone, battle barks, UI text, naming rules.
