# Lore

Use this folder for worldbuilding that supports the game without bloating source code.

## Suggested files

- `world.md` - high-level setting overview.
- `towns.md` - towns, hubs, shops, inns, routes, and faction control.
- `guardians.md` - Guardian mythology, corruption rules, unlock lore.
- `factions.md` - Null Conclave, Resonants, kingdoms, guilds, religious orders.
- `timeline.md` - historical events and campaign chronology.
