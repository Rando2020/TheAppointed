# Mechanics

Use this folder to define combat and progression rules before or alongside implementation.

## Suggested files

- `combat-loop.md` - ATB, turns, targeting, wave flow, win/loss conditions.
- `surge-deflect.md` - timing windows, input rules, bonus damage, defensive timing.
- `temper-ether.md` - armor logic, status resistance, break states.
- `elements-combos.md` - elements, surface reactions, combo chains.
- `jobs-progression.md` - base jobs, ascended jobs, JP economy, unlock requirements.
- `items.md` - consumables, shop logic, inventory rules.
