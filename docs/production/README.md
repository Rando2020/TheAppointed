# Production

Use this folder for project management, release planning, QA notes, and technical decisions.

## Suggested files

- `roadmap.md` - milestone plan from prototype to vertical slice.
- `decision-log.md` - important architecture/gameplay decisions and why they were made.
- `qa-checklist.md` - test scenarios for combat, UI, builds, and deployment.
- `backlog.md` - plain-English feature backlog before issues are created.
