# Assets

This folder tracks game art, audio, and generated placeholders.

## Pathways

- `concept/` - moodboards, references, approved art direction.
- `placeholders/` - temporary art/audio used in the playable prototype.
- `characters/` - sprites, portraits, class visuals, enemy sheets.
- `environments/` - maps, tiles, town backgrounds, combat arenas.
- `ui/` - icons, buttons, HUD elements, menus, status badges.
- `audio/` - SFX, voice barks, UI feedback sounds.
- `music/` - loops, themes, stingers, exported stems.

## Rule

Large source files should use Git LFS. Keep final naming stable and document generated prompts in `/prompts`.
